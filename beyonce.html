year: 2016
medium: digital tool, book
Processing, Captionbot, digital print

The project is a part of series of work created with the help of image recognition software called CaptionBot. Working in Processing, I wrote a colortracking code that generated drawings based on movement. To experiment with the tool, I used it on Beyonce music videos, and got a number of dancing trajectory images, that were analyzed and named by CaptionBot afterwards.
